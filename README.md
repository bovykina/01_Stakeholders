# Заинтересованные стороны (Стейкхолдеры)
## Описание
В рамках учебного проекта я изучала, кто такие заинтересованные стороны (стейкхолдеры) и как с ними работать в проектах. Задачи включали выявление стейкхолдеров, классификацию по ролям и слоям луковичной диаграммы, определение их интересов, потребностей и проблем, а также ведение каталога и глоссария. Цель проекта — научиться определять людей и организации, на которых влияет система, и эффективно формулировать требования.

## Полученные навыки
- Выявление стейкхолдеров и распределение их по ролям и категориям  
- Создание и ведение каталога заинтересованных сторон с ключевыми характеристиками  
- Построение луковичной диаграммы для визуализации взаимодействия стейкхолдеров с системой  
- Определение интересов, потребностей и проблем стейкхолдеров для формирования требований  
- Дополнение глоссария терминами, связанными с управлением стейкхолдерами  

## Этапы проекта
1. Создание каталога стейкхолдеров: сбор информации, определение характеристик и ролей  
2. Построение луковичной диаграммы с распределением стейкхолдеров по слоям и указанием ролей  
3. Анализ интересов, потребностей и проблем ключевых стейкхолдеров  
4. Наполнение глоссария новыми терминами и понятиями, связанными с проектом  
5. Проверка и актуализация всех артефактов для дальнейшей работы над системой  

## Заключение
Проект помог понять, как работать с заинтересованными сторонами, выявлять их потребности и распределять роли. Полученные навыки помогают строить требования, визуализировать взаимодействия и поддерживать актуальность информации о стейкхолдерах на протяжении всего проекта.
